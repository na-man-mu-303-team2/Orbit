# Editor asset sources

## Interface icons

- Runtime source: `@tabler/icons-react`
- License: MIT

## Presentation arrow shapes

- Runtime source: `presentationArrowAssets.tsx`
- License: original ORBIT source code
- Usage: editable-color SVG shapes with slide-ready aspect ratios for process, timeline, cycle, split, merge, and growth diagrams.
- Reference sites reviewed for visual coverage only: Flaticon arrow icons, Freepik gradient arrow vectors, SlidesCarnival arrow templates, and Miricanvas arrow presentation templates.

No SVG, PNG, HTML, or template content from those reference sites is copied into this repository. Their free and template results can require attribution, an account, or usage terms that do not clearly permit redistribution as an embedded product asset library.
